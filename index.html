🚗 O que é Claude Cowork

🚲
Chat

Conversa avulsa no claude.ai. Sem projeto, sem instruções persistentes. Tarefa única.

🚗
Cowork (Projects)

Chats agrupados num projeto. Contexto grande, knowledge files, custom instructions. Workflow recorrente.

🏎️
Code

Agente autônomo no terminal. Lê arquivos, edita código, executa comandos. Multi-passo.

Regra de bolso: use Cowork quando colar o mesmo prompt 3+ vezes, quando precisar carregar os mesmos arquivos toda hora, ou quando quiser que toda conversa começe do mesmo ponto.
