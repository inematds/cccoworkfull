🌊 O que é drift

Knowledge desatualiza, instructions acumula contradição, capacity satura. Resultado: respostas pioram ao longo do tempo sem ninguém perceber.

🛑 Sintomas de drift

  • • Respostas citam info que mudou meses atrás
  • • Tom/formato escapa do padrão definido
  • • Modelo ignora regras do instructions
  • • Tempo de resposta sobe; capacity bar acima de 80%
  • • Time para de usar — "Cowork não tá mais bom"
2

🗓️ Calendário de revisão

Cadência fixa transforma manutenção em hábito, não em emergência.

Q

Quinzenal

Olha capacity bar, remove arquivos que ninguém referenciou. ~10 min.

M

Mensal

Revisar instructions: contradições, regras inúteis, ajustes de tom. Roda baseline. ~30 min.

T

Trimestral

Revisão completa do projeto. Knowledge fresh, instructions atualizado, baseline novo, ROI medido. ~1-2h.

3

🧪 Baseline de qualidade

5-10 perguntas-prova com respostas esperadas. Rodar a cada mudança. Mudou e piorou? rollback.

BASELINE PARA "REVISÃO PR":

Q1: PR com SQL string-concat
    → Esperado: [ALTA] SQL injection, propor parametrização

Q2: PR adicionando função sem testes
    → Esperado: [MEDIA] cobertura, sugerir test cases

Q3: PR mudando estilo de naming
    → Esperado: ignorar (linter resolve), focar em lógica
4

📝 Changelog externo

Cowork não versiona instructions nativamente. Você precisa de audit trail manual: Google Doc/Notion com cada mudança.

2026-05-02 (Ana) — Adicionei restrição: não comentar estilo.
  Motivo: linter já cobre, comentários estavam ruidosos.

2026-04-15 (Bruno) — Mudei tom de "considere" pra "afirme/se cale".
  Motivo: hedge words estavam confundindo dev novato.

2026-04-01 (Ana) — Knowledge: trocou style-guide v1 por v2.
  Motivo: novas regras de async/await.

👤 Quem mantém

Owner declarado em cada projeto compartilhado. Sem owner, "responsabilidade de todos = de ninguém". Projeto morre.

💡 DRI por projeto

  • • Owner aparece no nome ou na descrição
  • • Owner é o responsável final, não único contributor
  • • Sucessão clara: se owner sai, transfere antes
  • • Owner roda a cadência (quinzenal/mensal/trimestral)
